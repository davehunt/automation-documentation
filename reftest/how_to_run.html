Running Reftests

Read first

  • Reftest overview

Desktop

With build environment

  1. Setup a Firefox build environment.
  2. Run make -C $(OBJDIR) reftest
    • $(OBJDIR) is the build's object directory, which is prefixed with obj-. The exact name depends on your system and OS.
    • To run a particular set of reftests, use TEST_PATH=path/from/sourcedir/reftest.list make -C $(OBJDIR) reftest
Automation currently only runs layout/reftests/reftest-sanity/reftest.list! If you try to run the full suite, you may experience stalls or other issues.

With built packages

  1. Download and extract the built test packages.
  2. Open a terminal and change to the directory of the extracted tests.
  3. Run python runreftest.py tests/layout/reftests/reftest-sanity

Mobile

Reftest does not support running tests on mobile.
